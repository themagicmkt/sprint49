# Sprint 49 — Landing Page

Página de vendas do **Sprint 49** — produto de entrada da [49 Educação](https://49educacao.com.br).
Em 7 dias o founder valida sua ideia no mercado real, ou descobre cedo que precisa pivotar.

## Stack

- HTML estático + Tailwind CSS (CDN)
- Fontes: Iceland (display) · Montserrat (corpo)
- Single-file, sem build

Hospedado em [Vercel](https://vercel.com). Cada push em `main` dispara deploy automático.
